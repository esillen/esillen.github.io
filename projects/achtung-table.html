Achtung die Kurve arcade Table

The Achtung die Kurve table is an arcade game based on the classic game “Achtung die Kurve” (a.k.a. Zatacka). It was designed, programmed and built by me with a special version of the game that allows for 8 players and some special rules.

The code for the game is open-source and can be found on Github

The Achtung Table itself is built out of wood, has 16 small pushbuttons (2 for each of the 8 players) and a large “start” pushbutton, a coin collecting mechanism, a lcd monitor and is driven by a raspberry pi. It is connected via wifi so it can be updated wireless.

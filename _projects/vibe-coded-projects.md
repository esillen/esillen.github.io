# Vibe coded projects

### Giving ideas I had piled up a shot using mostly vibe coding

- [Voice controlled pong (online multiplayer)](https://multiplayer-voice-game.fly.dev/) - Let the most beautiful (?) voice win!
- [Beach Assault](https://esillen.github.io/beach-assault/) - Take the beach using human wave tactics!
- [Clarinet Utils](https://esillen.github.io/clarinet-utils/) - Small tools for clarinet practice.
- [Co-op Tetris](https://esillen.github.io/coop-tetris/) - Play tetris together in the same space.
- [Jump](https://esillen.github.io/jump/) - Jump on other rabbits to win. Now with child mode for less gore!
- [Spacecraft](https://esillen.github.io/Spacecraft/) - Control one thruster out of up to 8! You will probably crash.
- [CrotchetCounterAndroidApp](https://github.com/esillen/CrotchetCounterAndroidApp) - Android app for crotchet counting using your feet and a video game controller.
- [Mages](https://esillen.github.io/mages/) - Spellcaster action game.

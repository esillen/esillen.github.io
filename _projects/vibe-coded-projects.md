# Vibe coded projects

- [Clarinet Utils](https://esillen.github.io/clarinet-utils/) - Practical utility tools for clarinet practice.

- [Jump](https://esillen.github.io/jump/) - Minimal, reflex-based jumping game.
- [Spacecraft](https://esillen.github.io/Spacecraft/) - Space-themed arcade gameplay experiment.
- [CrotchetCounterAndroidApp](https://github.com/esillen/CrotchetCounterAndroidApp) - Android app for rhythm/crotchet counting practice.
- [Mages](https://esillen.github.io/mages/) - Magic-themed action/strategy game experiment.
